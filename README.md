## üèÅ Getting Started <a name = "getting_started"></a>

- clone project
  ```
  git clone https://github.com/novicexp/bank-management-system
  ```
  ```
  cd bank-management-system/
  ```
- compile
  ```
  g++ -o bank bank.cpp
  ```
- run
  ```
  ./bank
  ```
